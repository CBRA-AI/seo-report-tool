# SEO Report Tool 🚀

This tool generates SEO reports using:

- Google Analytics
- Google Search Console
- Keyword ranking data

## Features

- Traffic overview (Users, Sessions)
- Keyword ranking (Position + Page URL)
- Charts & graphs
- Downloadable report (PDF - upcoming)

## Tech Stack

- Backend: Node.js (Express)
- Frontend: HTML, CSS, JavaScript
- Hosting: Render + Netlify

## Setup

1. Install dependencies:
   npm install

2. Start server:
   npm start

## Note

- Do not upload service-account.json
- Use environment variables for API keys
